## 記事本文（冒頭部分）
{{content}}

## 生成ルール
- 5〜8問のQ&Aを生成してください
- 記事の内容を補完する質問を含めてください
- SEO（AIO）に効果的なキーワードを自然に含めてください
- 回答は簡潔（2-3文）にしてください
- 最新のGoogle検索データを参照して、実際にユーザーが検索しそうな質問にしてください
- 「〜とは？」「〜の違いは？」「〜のメリットは？」などの検索意図を考慮してください

## 出力形式（JSON）
```json
{
  "faqs": [
    { "question": "質問文", "answer": "回答文" }
  ]
}
```

JSON以外のテキストは不要です。
